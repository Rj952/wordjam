# WORDJAM

**Jamaica's Creole-Affirming Literacy Platform**

> "Every word. Every voice. Every yard."
>
> Bridging Jamaican Patwa and Standard English for learners from Early Childhood through Grade 11. Culturally rooted. AI-powered. NSC-aligned.
>
> ## Features
>
> - **Creole Bridge Reader** - Toggle between Jamaican Patwa and Standard English
> - - **AI Writing Coach** - Claude-powered feedback that honours code-switching
>   - - **Oral Literacy Lab** - Record and analyse Anansi retellings with AI
>     - - **Jamaica Journey Map** - Gamified badges and progress tracking
>       - - **Teacher Command Centre** - NSC-aligned assignments, dashboards, AI review queue
>         - - **Family Literacy Portal** - Parent summaries and home reading activities
>          
>           - ## Tech Stack
>          
>           - - **Frontend:** Next.js 16, React, TypeScript
>             - - **AI Backend:** 7 API routes powered by Claude (Anthropic)
>               - - **Database:** Supabase (PostgreSQL) - schema in /schema/
>                 - - **Deployment:** Vercel
>                  
>                   - ## Quick Start
>                  
>                   - ```bash
>                     npm install
>                     npm run dev
>                     ```
>
